# 05 - Two-Week Comparison

Goal: run Langfuse and Phoenix against real retrieval-nord traffic/tests for two weeks.

## Daily Notes

Track these every day:

- What broke in instrumentation.
- Which questions the tools answered.
- Which questions still required manual reconstruction.
- Whether any quality degradation would have paged a human.
- Whether prompt version, evaluator version, and session outcome were visible.

## Exit Artifact

Write `SPEC.md` from unanswered operational questions:

- Is quality degrading?
- Should someone be paged?
- Which prompt version caused this?
- Which sessions are impacted?
- What action is safe?
- What should be held for human approval?
