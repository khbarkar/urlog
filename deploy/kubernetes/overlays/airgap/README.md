# Air-Gapped Overlay

Use this overlay when images and dependencies are mirrored into a private registry.

Before applying:

1. Mirror Urlog images into the customer registry.
2. Mirror any dependency images.
3. Configure the secret backend through local SOPS/age, Vault/OpenBao, or the customer's approved secret manager.
4. Replace image registry names in `kustomization.yaml`.

No default passwords are generated by these manifests.
